Chạy scan

Nhập target, chọn platform hoặc để auto, sau đó bấm bắt đầu fuzzing. Tool sẽ chuyển sang màn hình tiến độ để hiển thị quá trình scan.

Luồng xử lý của tool

Bước Mô tả
1 Fingerprint platform và server technology
2 Tạo wordlist theo ngữ cảnh platform, server extension và parameter type
3 Fuzzing path và parameter bằng canary payload an toàn
4 Phân tích phản hồi server để kết luận nguy cơ bug
5 Sinh JSON report và Markdown report
